Availity provides two supplementary APIs alongside the Service Reviews API for ASC X12N 278 authorizations and referrals: IsAuthRequired, which lets providers check whether a payer requires authorization, and Attachments, which lets third parties submit supporting documents. Both are currently unavailable to new customers.
